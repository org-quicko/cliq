# Introduction

Cliq is a referral and affiliate management platform designed for businesses to create, manage, and track referral programs. The API-first design ensures seamless integration with your existing systems, making it easy to automate workflows and manage referral data in real time.

## Features

* <b>Self-Hosted</b><br/>
     Deploy Cliq on your own servers for full control over your data and security.
* <b>RESTful API</b><br/>
     Integrate Cliq with your existing systems using its REST API, allowing automated referral management, real-time tracking, and reporting.
* <b>Multi-tenant</b><br/>
     Manage multiple programs, promoters, and members from a single platform with dedicated API endpoints for each.
* <b>Referral Tracking</b><br/>
     Track signups, purchases, and referrals across promoters and programs in real-time.
* <b>Commission Management</b><br/>
     Define and track commissions for promoters with detailed analytics and reporting.
* <b>Role-Based Access Control</b><br/>
     Define user roles and permissions for API access, ensuring secure and controlled access to program data.
