# Introduction

Cliq is a referral and affiliate management platform designed for businesses to create, manage, and track referral programs.

## Features

* <b>Self-Hosted</b><br/>
     Deploy Cliq on your own servers for full control over your data and security.
* <b>RESTful API</b><br/>
     Integrate Cliq with your existing systems using its REST API, allowing automated referral management, real-time tracking, and reporting.
* <b>Multi-tenant</b><br/>
     Manage multiple programs, promoters, and members from a single platform.
* <b>Referral Tracking</b><br/>
     Track signups, purchases, and referrals across promoters and programs. 
* <b>Role-Based Access Control</b><br/>
     Define user roles and permissions for access, ensuring secure and controlled access to program data.
